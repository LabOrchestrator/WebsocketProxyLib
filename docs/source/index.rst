Welcome to WebsocketProxyLib's documentation!
===================================================

**WebsocketProxyLib** is a Python library that adds JWT authentication to KubeVirts VNC Websockets.
It is used in the `WebsocketProxy <https://github.com/LabOrchestrator/WebsocketProxy>`_.

You can find the releases of the library at `pypi.org/project/lab-orchestrator-ws-proxy-lib/ <https://pypi.org/project/lab-orchestrator-ws-proxy-lib/>`_.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
